## Navigation

| Shortcut | Action |
| --- | --- |
| <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>←</kbd> | Switch to previous tab |
| <kbd>⌘</kbd> + <kbd>⌥</kbd> + <kbd>→</kbd> | Switch to next tab |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>C</kbd> | Open calendar |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>O</kbd> | Open contacts |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>,</kbd> | Open AI settings |
| <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>L</kbd> | Open folders |
| <kbd>⌘</kbd> + <kbd>,</kbd> | Open settings |
| <kbd>⌘</kbd> + <kbd>O</kbd> | Open note dialog |
| <kbd>⌃</kbd> + <kbd>⌥</kbd> + <kbd>←</kbd> | Switch to previous panel tab |
| <kbd>⌃</kbd> + <kbd>⌥</kbd> + <kbd>→</kbd> | Switch to next panel tab |


## Sidebar & Panels

| Shortcut | Action |
| --- | --- |
| <kbd>⌘</kbd> + <kbd>J</kbd> | Toggle chat panel |
| <kbd>⌘</kbd> + <kbd>\\</kbd> | Toggle sidebar |
| <kbd>Esc</kbd> | Dismiss / close |
